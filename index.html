Welcome to Porcupine’s plugin API documentation!¶

This documentation is all about writing your own Porcupine plugins in Python to make Porcupine do something fun that it doesn’t do by default.

If you write a nice plugin, please share it with me! I’m interested to see what you have made. Just create a pull request on GitHub that adds your plugin to porcupine/plugins or more_plugins.

The Tutorial¶

Start by reading this if you’re new here. Among other things, it contains a hello world example with detailed instructions for installing the plugin, but it also contains a list of things that you need to be careful with or watch out for.

API Reference¶

Everything that you can use in plugins is documented here.

  • porcupine — The top level Porcupine package
  • porcupine.menubar — The menu bar at the top of the Porcupine window
  • porcupine.tabs — The tab manager and tabs in it
  • porcupine.textutils — Things for tkinter.Text widgets
  • porcupine.images — Load images that come with Porcupine
  • porcupine.settings — Porcupine’s setting manager
  • porcupine.pluginloader — The module that loads plugins on startup
  • porcupine.utils — Handy utility functions and classes
